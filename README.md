# DMTshow — ComedyCollab

Комедийное шоу ComedyCollab. Веб-приложение для регистрации участников с панелью администратора.

## Стек

- **Frontend:** React + Vite, styled-components, framer-motion, axios
- **Backend:** Node.js + Express

## Запуск

### 1. Установка зависимостей

```bash
npm install
```

### 2. Запуск бэкенда

```bash
ADMIN_PASSWORD=yourpassword npm run server
```

> **Важно:** переменная окружения `ADMIN_PASSWORD` обязательна для работы панели администратора.
> Данные хранятся в памяти сервера — при перезапуске обнуляются.

### 3. Запуск фронтенда (режим разработки)

```bash
npm run dev
```

Откройте [http://localhost:5173](http://localhost:5173).

### 4. Сборка для продакшена

```bash
npm run build
```

## API

| Метод | Путь | Описание |
|-------|------|----------|
| GET | `/api/count` | Число зарегистрированных участников |
| POST | `/api/register` | Регистрация участника |
| POST | `/api/admin/login` | Вход в панель администратора |
| GET | `/api/participants` | Список участников (требует авторизации) |

## Переменные окружения

| Переменная | Описание | Обязательная |
|------------|----------|--------------|
| `ADMIN_PASSWORD` | Пароль для входа в панель администратора | Да |
| `PORT` | Порт сервера (по умолчанию 3001) | Нет |
